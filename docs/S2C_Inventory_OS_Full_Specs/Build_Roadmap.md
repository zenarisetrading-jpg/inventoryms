# Build Roadmap — S2C Inventory OS

## MVP Target
5 weeks internal MVP, 8–10 weeks production Phase 1.

## Week 0 — Foundation
Architecture lock, schema, repos.

## Week 1 — Data Layer
Amazon + Locad integration, Noon upload.

## Week 2 — Demand + PO Module
Sales velocity engine + PO Register.

## Week 3 — Decision Engine
Coverage, allocation, reorder, transfer.

## Week 4 — Operator UI
Command Center, SKU Detail, PO Register screens.

## Week 5 — Testing
Integration, QA, edge cases.

## Weeks 6–10 — Hardening
Performance, workflow, planning view, automation hooks.
